# User stories

## US1
As a user, I want to be able to view a map of the UK,so that I can select an area to see if products are present.

## US2
As I user, I would like to be able to assess how coverage has built up over time.

## US3 
As a user, I would like to be able to view coverage of products as a percentage of UK land area

## US4
As a user, I would like to be able to view coverage of products as a percentage of an area specifed by me, so I can assess data in otherwised undefined areas.

## US5 
As a user, I would like to be able to view percentage coverage by entering in an area e.g. 'Sheffield' or 'Manchester', so that I can assess data in defined regions.
